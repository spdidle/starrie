# starlight 4 stasis

This is my very first PCB board, that I made star shaped following the beginner projects page on [Stasis](https://stasis.hackclub.com/starter-projects). This project uses an NE555P timer chip in astable mode to generate a clock pulse, which drives a CD4017 decade counter to sequence 10 LEDs.
